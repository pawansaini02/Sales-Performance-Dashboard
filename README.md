## 🎯 Project Overview

This project simulates a real-world BI Analyst workflow for a US-based consulting company. It covers the full pipeline:

1. **Database design** — normalized relational schema in MySQL/PostgreSQL
2. **SQL analytics** — KPI queries, funnel analysis, cohort trends, win rate by rep
3. **Interactive dashboard** — live HTML/JS dashboard with filters and drill-down
4. **Automated reporting** — Python script that generates a formatted Excel report and emails it every Monday

## 🚀 How to Run Locally

### Option A — Just view the dashboard
```bash
# Clone the repo
git clone https://github.com/your-username/vrt-sales-dashboard.git
cd vrt-sales-dashboard

# Open in browser — no server needed
open index.html
```

### Option B — Run the full pipeline
```bash
# Install dependencies
pip install pandas openpyxl mysql-connector-python

# Generate sample CSV data
python python/generate_sample_data.py

# Import SQL schema
mysql -u root -p < sql/01_create_schema.sql

# Generate Excel report
python python/generate_report.py
```

### Option C — Connect to a live database
1. Set environment variables:
```bash
export DB_HOST=localhost
export DB_NAME=vrt_sales
export DB_USER=analyst
export DB_PASS=your_password
```
2. Load sample data into MySQL using `data/deals.csv`
3. Run `python/generate_report.py`
